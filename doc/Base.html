Class: Base

Base

Javascript KTBS.Base Object that is bound to a KTBS.

Constructor

new Base(uri, idopt)

Samotraces.KTBS.Base is a Javascript KTBS base object that is bound to a KTBS. This Object implements the KTBS API. Methods are available to get the list of traces available in the KTBS base. Access a specific trace, etc.
