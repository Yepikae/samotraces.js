Class: Timer

Timer

Object that stores the current time

new Timer(init_time, period, update_function)

Object that stores the current time

Samotraces.Lib.Timer is a Javascript object that stores the current time. This Object stores a time and informs widgets or other objects when the time changes.

Parameters:
Name Type Argument Default Description
init_time Number <optional>
 0

Initial time of the timer (optional, default: 0).
period Number <optional>
 2000

Perdiod (in ms) at which the timer will update itself in "play" mode.
update_function function <optional>

Function called to update the timer when in "play" mode (function that returns the value of Date.now() by default).
