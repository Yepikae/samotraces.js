TimePlayer

Constructor

new TimePlayer(html_id, timer, videosopt)

Widget for playing/pausing a timer and controlling videos.

Samotraces.UI.Widgets.TimePlayer is a Widget that allow to trigger the "play/pause" mechanism of a timer. In addition, it controls a set of videos that are synchronised to this timer.

This widget observes a Samotraces.Timer object. When the timer changes the videos are . This widget also allow to change the time of the timer.

Name Type Attributes Description
html_id String

Id of the DIV element where the widget will be instantiated
timer Samotraces.Timer

Timer object to observe.
videos Array.<Samotraces.UI.Widgets.TimePlayer.VideoConfig> <optional>

Array of VideoConfig, that defines the set of videos that will be synchronised on the timer.
