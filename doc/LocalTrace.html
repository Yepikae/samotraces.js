LocalTrace

Constructor

new LocalTrace()

Javascript Trace Object.

Samotraces.DemoTrace is a Javascript Trace object. Methods are available to get the Obsels from the trace, create new Obsels, etc.

The trace is initialised empty. Obsels have to be created by using the Samotraces.DemoTrace#newObsel method.

list_obsels(beginopt, endopt, reverseopt) → {Array.<Obsel>}

Returns the list of obsels in an optional time interval. If no minimum time and no maximum time constraint are defined, returns the whole list of obsels. If one of the two constraints are defined, then returns obsels matching the time constraints.

Note: if an obsel overlaps with the start or the end constraint, then it will be included (for instance an obsel that starts before the start constraint and ends after that constraint will be included).

Parameters:
Name Type Attributes Default Description
begin Number <optional>

Minimum time constraint
end Number <optional>

Maximum time constraint
reverse Boolean <optional>
 false

Returns the obsel list in reverse chronological order if true and in normal chronological order if false.
