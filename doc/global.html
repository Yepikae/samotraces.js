ObselParam

ObselParam is an object that contains parameters necessary to create a new obsel. This type of object is used in several methods such as the Obsel constructor, or the Trace.create_obsel method. The optional porperties varies depending on the method called.

Properties:
Name Type Argument Description
id String <optional>

Id of the obsel
trace Trace <optional>

Trace of the obsel
type String <optional>

Type of the obsel
begin Number <optional>

Timestamp of when the obsel starts
end Number <optional>

Timestamp of when the obsel ends
attributes Object <optional>

Attributes of the obsel.
relations Array.<Relation> <optional>

Relations from this obsel.
inverse_relations Array.<Relation> <optional>

Relations to this obsel.
source_obsels Array.<Obsel> <optional>

Source obsels of the obsel.
param.label String <optional>

Label of the obsel.
