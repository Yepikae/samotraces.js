Class: Trace

Samotraces.Lib. Trace

JavaScript (abstract) Trace class.

<virtual> new Trace()

JavaScript (abstract) Trace class.

Samotraces.Lib.DemoTrace is a Javascript Trace object. Methods are available to get the Obsels from the trace, create new Obsels, etc.

The trace is initialised empty. Obsels have to be created by using the Samotraces.Lib.DemoTrace#newObsel method.
