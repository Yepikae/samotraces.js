Class: ReadableTimeForm

Samotraces.UI.Widgets. ReadableTimeForm

Widget for visualising the current time as a date/tim.

new ReadableTimeForm(html_id, timer)

Widget for visualising the current time as a date/time.

Samotraces.UI.Widgets.ReadableTimeForm is a generic Widget to visualise the current time.

The time (in ms from the 01/01/1970) is converted in a human readable format (as opposed to Samotraces.Widgets.TimeForm widget which display raw time).

This widget observes a Samotraces.Lib.Timer object. When the timer changes the new time is displayed. This widget also allow to change the time of the timer.

Parameters:
Name Type Description
html_id String

Id of the DIV element where the widget will be instantiated
timer Samotraces.Timer

Timer object to observe.
