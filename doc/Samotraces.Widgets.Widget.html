Methods

add_behaviour(behaviourName, eventTargetElement, opt)

Creates a new behaviour (interaction possibility) with the widget. Two behaviours are implemented so far: 1. 'changeTimeOnDrag' 2. 'zommOnScroll'

  1. 'changeTimeOnDrag' behaviour allows to change a Samotraces.Lib.Timer on a drag-n-drop like event (JavaScript 'mousedown', 'mousemove', 'mouseup' and 'mouseleave' events). This allows to change the current time by dragging a trace visualisation or a slider for instance.

  2. 'changeTimeOnDrag' behaviour allows to change a Samotraces.Lib.TimeWindow on a mouse scroll event (JavaScript 'wheel' event)

Parameters:
Name Type Description
behaviourName String

Name of the behaviour ('changeTimeOnDrag' or 'zommOnScroll'). See description above.
eventTargetElement HTMLElement

HTML Element on which an eventListener will be created (typically, the element you want to interact with).
opt Object

Options that vary depending on the selected behaviour.

Properties
Name Type Description
onUpCallback function

(for 'changeTimeOnDrag' behaviour only) Callback that will be called when the 'mouseup' event will be triggered. The argument delta_x is passed to the callback and represents the offset of the x axis in pixels between the moment the mousedown event has been triggered and the moment the current mouseup event has been triggered.
onMoveCallback function

(for 'changeTimeOnDrag' behaviour only) Callback that will be called when the 'mousemove' event will be triggered. The argument delta_x is passed to the callback and represents the offset of the x axis in pixels between the moment the mousedown event has been triggered and the moment the current mousemove event has been triggered.
timeWindow Samotraces.Lib.TimeWindow

(for 'zommOnScroll' behaviour only) Samotraces.Lib.TimeWindow object that will be edited when the zoom action is produced.
