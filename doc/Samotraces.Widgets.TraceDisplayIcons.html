Class: TraceDisplayIcons

Samotraces.Widgets. TraceDisplayIcons

Widget for visualising a trace where obsels are displayed as images

new TraceDisplayIcons(divId, trace, time_window, options)

Widget for visualising a trace where obsels are displayed as images.

The TraceDisplayIcons widget is a generic Widget to visualise traces with images. This widget uses d3.js to display traces as images in a SVG image. The default settings are set up to visualise 16x16 pixels icons. If no url is defined (see options), a questionmark icon will be displayed by default for each obsel.

Note that clicking on an obsel will trigger a ui:click:obsel event.

Tutorials Tutorial 1 – part 1: Creating a trace visualisation in 10 minutes, Tutorial 1 – part 2: Adding widgets to an application, and Tutorial 1 – part 3: Putting it all together and personalising the visualisation illustrates in more details how to use this class.

Properties
Name Type Argument Description
visu VisuConfig <optional>

Object determining how to display the icons (Optional). All the options field can be either a value or a function that will be called by d3.js. The function will receive as the first argument the Obsel to display and should return the calculated value. If a function is defined as an argument, it will be binded to the TraceDisplayIcons instance. This means that you can call any method of the TraceDisplayIcons instance to help calculate the x position or y position of an icon. This makes it easy to define various types of behaviours. Relevant methods to use are: link Samotraces.Widgets.TraceDisplayIcons.calculate_x} See tutorial Tutorial 1 – part 3: Putting it all together and personalising the visualisation for more details and examples.
events EventConfig <optional>

Events to listen to and their corresponding callbacks.

Example
var options = {
    visu: {
        y: 20,
        width: 32,
        height: 32,
        url: function(obsel) {
            switch(obsel.type) {
                case 'click':
                    return 'images/click.png';
                case 'focus':
                    return 'images/focus.png';
                default:
                    return 'images/default.png';
            }
        }
    },
    events: {
        'ui:click:obsel': obsel_selector.select.bind(obsel_selector)
    }
};
