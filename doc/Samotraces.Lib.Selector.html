Class: Selector

Samotraces.Lib. Selector

Object that stores a selection of objects

new Selector(type, selection_mode, events)

Object that stores a selection of objects

The Selector object is a Javascript object that stores a selection of Objects. This Object stores Objects that are selected and informs widgets or other objects (via the triggered events) when the selection changes. When first instanciated, the selection is empty.

In order to select an object, the Selector#select() method has to be called. Similarly, in order to unselect an object, the Selector#unselect() method has to be called. The whole selection can be emptied at once with the Selector#empty() method.

Parameters:
Name Type Argument Default Description
type string

A string describing the type of object to be selected ('Obsel', 'Trace', 'TimeWindow', etc.).
selection_mode string <optional>
 'single'

In 'single' mode, the selection contains one object maximum. This means that adding an object to a non-empty selection will replace the previously selected object with the new one. In 'multiple' mode, the selection can be extended and objects can be individually added or removed.
events EventConfig <optional>

Events to listen to and their corresponding callbacks.
