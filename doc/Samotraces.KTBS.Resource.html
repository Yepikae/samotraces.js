Class: Resource

Samotraces.KTBS. Resource

Resource Objects have an uri, an id and an optional label

new Resource(id, url, type, label)

Resource Objects that is synchronised to a kTBS

Resource Objects are KTBS objects. All resources have an uri, an id and an optional label
